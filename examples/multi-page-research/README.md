# Multi Page Research

Use this example when the task spans multiple official public sources and needs replayable evidence.

```bash
pnpm run pilot:public-reference-workflow

pnpm run pilot:real-user-research
```

What this shows:

- MCP-driven multi-tab research
- evidence extraction across several public domains
- session synthesis and replay-ready state
- where `touch-browser` is stronger than a single-page markdown fetch tool
